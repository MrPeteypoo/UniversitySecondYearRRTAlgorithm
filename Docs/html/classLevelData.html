Reads the header of a map file and prepares the tile data accordingly. Throws exceptions if the header is invalid.

Parameters
streamAn open file stream to read from.

The header is in the following format: "type blah", ignore this. "height blah", gives us the height value. "width blah", gives us the width value. "map", ignore this.
